# DrinkingBirdBot

Components:

Arduino Uno
Nema 23 Stepper Motors and Motor Mount L Brackets
Pololu Mounting Hub for 1/4" (6.35mm) Shaft
24V 5A Power Supply with 5.5x2.1mm DC Output Jack
MYSWEETY TB6600 0.2-5A Nema 17/23 Stepper Motor Driver
